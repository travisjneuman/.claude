## Research Workflow

### Before Any UI Work

```
1. DEFINE: What type of component/page/app?
2. RESEARCH: Search 3-5 sources from above
3. COLLECT: Save 5-10 relevant examples
4. ANALYZE: What patterns repeat? What's unique?
5. ADAPT: Extract principles, not pixels
6. DESIGN: Create with human touch
```

### Source Selection by Task

| Task | Primary Sources |
|------|-----------------|
| Landing page | Land-book, Godly, One Page Love |
| Dashboard | Dribbble (search), SaaS Landing Page |
| Mobile app | Mobbin, UI Sources, Screenlane |
| E-commerce | Awwwards, Baymard Institute |
| SaaS product | Mobbin, UI Sources, Land-book |
| Portfolio | Awwwards, One Page Love |
| Component design | shadcn/ui, Collect UI, Tailwind Plus |

### Quality Signals to Look For

**Good Inspiration:**
- Real shipped products (not concepts)
- Consistent design system
- Thoughtful micro-interactions
- Clear visual hierarchy
- Accessible color contrast
- Mobile-responsive

**Avoid:**
- Dribbble "shots" with no real-world constraints
- Over-designed conceptual work
- Designs that prioritize looks over usability
- Patterns from 2-3+ years ago

## Anti-AI-Look Checklist

When reviewing designs, ensure they don't have these "obviously AI" markers:

### Visual Red Flags
- [ ] Generic blue/purple gradient backgrounds
- [ ] Perfectly symmetrical layouts
- [ ] Stock illustration style (Undraw, etc.)
- [ ] Generic "hero with laptop mockup" patterns
- [ ] Overuse of glassmorphism/blur effects
- [ ] Cookie-cutter card layouts
- [ ] Default shadcn/ui gray color scheme

### Adding Human Touch
- [ ] Custom color palette (not defaults)
- [ ] Intentional asymmetry
- [ ] Unique typography pairing
- [ ] Custom illustrations or photography
- [ ] Micro-interactions with personality
- [ ] Subtle imperfections (organic shapes)
- [ ] Brand-specific details

### What AI Can't Do Well
- Cultural nuance and context
- Emotional design decisions
- Brand personality expression
- Edge case handling
- Accessibility beyond basics
- Performance optimization trade-offs
- User testing insights
