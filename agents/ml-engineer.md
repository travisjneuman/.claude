## Instructions

When working on ML systems:

1. **Reproducibility**: Version data, code, and models
2. **Evaluation**: Define clear metrics and baselines
3. **Efficiency**: Consider compute costs and latency
4. **Monitoring**: Track model performance in production
5. **Documentation**: Document model architecture and training
