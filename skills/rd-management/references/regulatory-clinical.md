# Regulatory & Clinical Development Reference

## Regulatory Strategy

```
REGULATORY PLANNING:

PRE-SUBMISSION:
- Regulatory mapping
- Classification determination
- Pathway selection
- Pre-submission meetings

SUBMISSION:
- Documentation preparation
- Clinical/technical data
- Quality system documentation
- Regulatory application

REVIEW:
- Agency interaction
- Information requests
- Deficiency responses
- Approval/clearance

POST-MARKET:
- Compliance monitoring
- Adverse event reporting
- Changes/supplements
- Renewals
```

## Industry-Specific Pathways

| Industry           | Primary Regulator    | Key Pathways             |
| ------------------ | -------------------- | ------------------------ |
| **Pharma**         | FDA, EMA             | NDA, BLA, 505(b)(2)      |
| **Medical Device** | FDA, Notified Bodies | 510(k), PMA, De Novo     |
| **Food**           | FDA, USDA            | GRAS, Food Additive      |
| **Chemical**       | EPA, REACH           | TSCA, REACH registration |
| **Automotive**     | NHTSA, EPA           | FMVSS, Emissions         |

## Clinical Development

```
CLINICAL TRIAL PHASES:

PRECLINICAL:
- Lab studies
- Animal studies
- Safety pharmacology
- IND/IMPD preparation

PHASE 1:
- Safety focus
- 20-100 subjects
- Dose finding
- PK/PD studies

PHASE 2:
- Efficacy signal
- 100-500 subjects
- Dose optimization
- Safety monitoring

PHASE 3:
- Pivotal efficacy
- 1000+ subjects
- Regulatory endpoints
- Safety database

PHASE 4:
- Post-market
- Real-world evidence
- Long-term safety
- Label expansion
```
