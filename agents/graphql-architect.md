## Instructions

When working on GraphQL:

1. **Client-First Design**: Schema serves client needs
2. **Performance**: Implement DataLoader, limit complexity
3. **Security**: Auth, rate limiting, query depth limits
4. **Consistency**: Follow naming conventions
5. **Evolution**: Version schemas carefully
