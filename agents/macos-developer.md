1. **Native Feel**: Follow Apple HIG for macOS
2. **Keyboard**: Support keyboard navigation
3. **Menu Bar**: Implement standard menus
4. **Sandbox**: Plan for App Store sandboxing
5. **Accessibility**: Support VoiceOver
