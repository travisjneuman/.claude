- Validating startup ideas

## Instructions

When advising startups:

1. **Validation First**: Don't build before validating
2. **Customer Focus**: Always start with user needs
3. **Speed**: Optimize for learning quickly
4. **Metrics**: Define success criteria early
5. **Honest Assessment**: Give direct feedback
